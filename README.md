# AI Writing Skills

This repository stores reusable AI Agent skills.

## Skills

### article-matrix-generator

用于根据项目资料、GitHub 链接、产品文档或文字描述，自动生成文章矩阵。

默认生成：
- 1 篇主页面文章
- 2 篇引流文章

要求：
- 严格基于资料
- 不胡编乱造
- 写明引用链接
- 区分已实现功能和应用设想
- 文章简洁、有逻辑、生动
